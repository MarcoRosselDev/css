# ¿Cuáles son los términos de diseño más comunes para comunicarte con los diseñadores? 

NOTA: Algunos ejemplos interactivos podrían usar CSS que aún no conoces. No te preocupes por intentar entender todo el código. El objetivo de los ejemplos es mostrarte una vista previa de estos conceptos de diseño para que comprendas mejor cómo funcionan.
Como desarrollador, es posible que tengas que trabajar en estrecha colaboración con diseñadores. Si comprendes la terminología básica del diseño, compartirás un vocabulario común y podrás participar más activamente en el proceso de diseño. Esto puede optimizar tu flujo de trabajo y mejorar la experiencia del usuario.
Así que, ¡manos a la obra! Empezaremos con el término maquetación. La maquetación se refiere a cómo se organizan los elementos visuales en una página o pantalla para comunicar un mensaje. Estos elementos pueden incluir texto, imágenes y espacios en blanco. La maquetación es como el plano de un diseño. Los diseñadores deben considerar la ubicación, el tamaño y la jerarquía de cada elemento.

[Link a sandbox para ver el diseño](https://codesandbox.io/p/sandbox/j44k3x?file=%2Findex.html%3A24%2C39)

El término estrechamente relacionado con el diseño es la alineación. La alineación se refiere a cómo se colocan los elementos entre sí. Usar la alineación correctamente ayuda a que el diseño se vea limpio y organizado. Los diseñadores crean orden visual alineando los elementos a lo largo de líneas imaginarias, bordes o un punto central.
